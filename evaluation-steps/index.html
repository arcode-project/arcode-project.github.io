Evaluation Steps

  1. The videos are available here.
  2. FFmpeg can be used to convert the video files to raw YUV files: ffmpeg -i input -c:v rawvideo -pix_fmt yuv420p output.yuv.
  3. FFmpeg further is used to extract the raw YUV frames: ffmpeg -f rawvideo -framerate 60 -s 1920x1080 -pixel_format yuv420p -i input.yuv -c copy -f segment -segment_time 0.01 frames%d.yuv.
  4. The original and reconstructed videos are not perfectly aligned. We wrote a code that aligns the reconstructed video frames with respect to those of the original one and calculates the PSNR.
