About ARCODE

ARCODE is proposed for massively multiplayer online role-playing games running on cloud gaming platforms. It captures different action data for different types of in-game objects and form an aerea of effect around the object. Based on the importance of each object to the player, considering the actions the player's avatar can perfom in each state of the game, and the actions the ohter obejects can perform during the gameplay, ARCODE determines how frequenntly the position of each in-game object should be updated. Reducing the number of motion updates of each object makes the encoder to use fewer bits to code the video frames in the cloud gaming platforms.
